# Ciclo do Negócio de Cibersegurança — Site da Formação (GitHub Pages)

Site de apoio à formação (ISCAC, 35h) · Formador: Nuno Salvador
Publicado em: https://guilhasn.github.io/ciclo_negocio_ciberseguranca/

## Estrutura
- `index.html` — hub com a cadência das 9 sessões
- `s1.html` — Sessão 1 (novas sessões: duplicar e ajustar)
- `quiz-m1.html` — quiz interativo do Módulo 1 (anónimo, client-side)
- `tabletop-dce.html` — exercício tabletop "Incidente no Fornecedor do DCE" (client-side; entrega por ficha Word)
- `fichas/` — fichas de trabalho Word para os formandos descarregarem

## Publicar (uma vez)
1. Repositório `ciclo_negocio_ciberseguranca` → Add file → Upload files → arrastar TODO o conteúdo desta pasta → Commit.
2. Settings → Pages → Branch: `main` / pasta `/ (root)` → Save.
3. O site fica em https://guilhasn.github.io/ciclo_negocio_ciberseguranca/ em ~1 minuto.

## Atualizar
Upload do ficheiro novo/alterado no repositório → Commit. O Pages republica sozinho.
